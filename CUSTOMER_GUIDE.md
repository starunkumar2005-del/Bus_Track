🌟 GETTING STARTED (Easy 3 Steps)
───────────────────────────────────────────────────────────

Step 1: Open the App
  → Go to your browser
  → Type: http://localhost:3000 (or your app URL)
  → Wait 2-3 seconds for map to load

Step 2: See All Buses
  → Map shows all 10 Tamil Nadu cities
  → 🚌 Bus icons = Current bus position
  → 🛑 Red markers = Bus stops
  → Different colored lines = Different routes

Step 3: Track Your Bus
  → Click on any bus icon (🚌)
  → See bus details in the panel

Top Information Bar:
  📍 10 Routes    = Total routes available
  🛑 50+ Stops    = Total bus stops across Tamil Nadu
  🚌 5 Buses      = Active buses running now

Filter Dropdown:

🚌 HOW TO TRACK A BUS
───────────────────────────────────────────────────────────

Method 1: Click Bus Icon on Map
  1. Find 🚌 (bus icon) on the map
  2. Click it
  3. See bus panel on bottom-right with:
     ✓ Bus Number (e.g., TN-01-AB-1001)
     ✓ Bus Type (AC / NON-AC / SEMI-AC)
     ✓ Current Status (running/stopped)
     ✓ Driver Name & Phone
     ✓ Passengers count
     ✓ Live GPS coordinates

Method 2: Filter by Route
  1. Use dropdown at top: "Filter Route"
  2. Select specific route (e.g., "101 - City Center Route")
  3. Map zooms to show that route
  4. See only buses on that route
  5. Click bus for details


🛑 UNDERSTANDING BUS STOPS
───────────────────────────────────────────────────────────

Stop Markers:
  • Red 🛑 markers = Bus stops on route
  • Click any red marker to see:
    ✓ Stop Name (e.g., Central Station)
    ✓ Route Number
    ✓ Stop Position (1st stop, 2nd stop, etc.)
    ✓ Wait Time at Stop (e.g., 3 seconds)

What Happens at Stops:
  1. Bus arrives at stop (wait 3 seconds)
  2. Passengers board/exit
  3. Bus departs to next stop
  4. Real-time update shows movement

⚠️ COMMON QUESTIONS & ANSWERS
───────────────────────────────────────────────────────────

Q: Why does the bus move?
A: Buses move along predefined routes every 2 seconds. This is simulated movement for demo purposes.

Q: How accurate is the location?
A: Location uses 7 decimal place precision (accurate to ~1 meter in real system).

Q: What if map is blank?
A: Try zooming out to see full Tamil Nadu. Map loads from 10km view.

Q: How many buses are tracked?
A: Currently 5 buses are simulated. In production, can track unlimited buses.

Q: Can I book a ticket through this?
A: No, this is tracking only. For booking, use separate ticketing system.

Q: Why do buses pause at stops?
A: Realistic simulation - buses wait 3 seconds at stops for passengers.

Q: What if my connection is slow?
A: Updates may lag 1-2 seconds. Check your internet speed.

Q: Can I see past route history?
A: Not in current version. System shows live tracking only.

🆘 TROUBLESHOOTING
───────────────────────────────────────────────────────────

Issue: Map won't load
Solution: 
  • Wait 5 seconds
  • Refresh page (F5)
  • Check internet connection
  • Try different browser

Issue: Buses not moving
Solution:
  • Check if real-time updates started
  • Look for live changes every 2 seconds
  • Refresh if needed
  • Check backend status

Issue: Can't click bus
Solution:
  • Make sure click on bus icon exactly
  • Not on the background
  • Try different bus marker
  • Close open panels first

Issue: Text too small on mobile
Solution:
  • Pinch zoom to enlarge
  • Browser zoom (Ctrl + Plus)
  • Rotate phone to landscape

Issue: Slow or laggy
Solution:
  • Check internet speed
  • Close other browser tabs
  • Clear browser cache
  • Restart app
