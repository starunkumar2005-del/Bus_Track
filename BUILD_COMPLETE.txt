📊 API ENDPOINTS (20+ Endpoints)
═══════════════════════════════════════════════════════════════════════════════

Routes:
  GET    /api/routes                 List all routes
  POST   /api/routes                 Create new route
  GET    /api/routes/:id             Get route details
  PUT    /api/routes/:id             Update route
  DELETE /api/routes/:id             Delete route

Buses:
  GET    /api/buses                  List all buses
  POST   /api/buses                  Create bus
  GET    /api/buses/:id              Get bus details
  GET    /api/buses/:id/location     Get real-time location
  POST   /api/buses/:id/assign-route Assign to route
  DELETE /api/buses/:id              Delete bus

Stops:
  GET    /api/stops                  List all stops
  POST   /api/stops                  Create stop
  GET    /api/stops/:id              Get stop details
  PUT    /api/stops/:id              Update stop
  DELETE /api/stops/:id              Delete stop

Schedules:
  GET    /api/schedules              List all schedules
  POST   /api/schedules              Create schedule
  GET    /api/schedules/:id          Get schedule details
  PUT    /api/schedules/:id          Update schedule
  DELETE /api/schedules/:id          Delete schedule

💡 CUSTOMIZATION
═══════════════════════════════════════════════════════════════════════════════

Adjust Bus Speed:
  Edit: server/.env
  Change: BUS_SPEED_KMH=50 (default: 40)

Adjust Update Frequency:
  Edit: server/.env
  Change: SIMULATION_UPDATE_INTERVAL=1000 (default: 2000ms)

Adjust Stop Duration:
  Edit: server/.env
  Change: STOP_DWELL_TIME_MS=5000 (default: 3000ms)

Adjust Traffic Variance:
  Edit: server/.env
  Change: TRAFFIC_DELAY_VARIANCE=0.3 (default: 0.2 = 20%)

Change Database:
  Edit: server/.env
  Change: MONGODB_URI= (use MongoDB Atlas connection string)


🚨 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: "Port 5000 already in use"
Solution: Edit server/.env, change PORT=5001

Problem: "MongoDB connection refused"
Solution: Run "mongod" OR use MongoDB Atlas cloud (see docs)

Problem: "Buses not moving"
Solution: Check backend console, reseed data, refresh page

Problem: "Frontend not loading"
Solution: Verify backend running, check browser console, F12

See detailed troubleshooting in:
  → QUICK_REFERENCE.md
